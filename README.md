## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

bash
git clone <repository-url>

2. Istall dependencies:

cd portfolio
npm install

3. Start the development server:

npm start

4. Open your browser and visit: `http://localhost:3000`
